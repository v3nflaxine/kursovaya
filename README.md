Эта программа представляет собой консольное приложение, моделирующее работу магнитной ленты как структуру данных с помощью двусвязного списка (Doubly Linked List) на языке C++. Каждая "запись" на ленте (узел списка) хранит информацию об одном файле.
Описание программы
Программа реализована в виде класса MagneticTape, который управляет двусвязным списком узлов TapeNode. Каждый узел содержит поля:
-filename (имя файла)
-sizeKb (размер файла в Кб)
-createTime (время создания)

Двусвязный список позволяет эффективно выполнять операции вставки и удаления как в начале/конце, так и в середине списка, а также обеспечивает удобный обход в обоих направлениях.
Программа предоставляет пользователю интерактивное меню со следующим функционалом:
Добавление записи в конец ленты (addToTail): Добавляет новую запись о файле в конец списка.
Вставка записи в произвольное место (insertAt): Вставляет новую запись по заданному пользователем индексу (позиции).
Вывод содержимого ленты (display): Отображает все записи, хранящиеся на ленте, с указанием номера, размера, времени создания и имени файла.
Удаление записи по позиции (deleteAt): Удаляет запись по заданному пользователем индексу.
Экспорт ленты в файл (exportToFile): Сохраняет текущее содержимое ленты в текстовый файл (.txt).
Импорт ленты из файла (importFromFile): Загружает записи из текстового файла, заменяя текущее содержимое ленты.
Поиск записи (search): Ищет записи по заданному полю: имени, размеру или времени создания.
Сортировка ленты (sort): Сортирует все записи на ленте методом "пузырька" (Bubble Sort) по выбранному полю: имени, размеру или времени создания.
Выход (0): Завершает работу программы.
